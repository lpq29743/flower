# flower

本工程是在自己大学阶段的《PHP 程序设计》一课上，利用 PHP7 + Laravel 5.4 完成的大作业。本项目实现了 Lavarel 5.4 的大部分功能，欢迎 star 和提 issue 和 PR，共同学习，共同进步！

## 前期准备

```bash
git clone https://github.com/lpq29743/flower.git
cd flower
mysql -u 用户名 -p 密码
source flowernew.sql
```
